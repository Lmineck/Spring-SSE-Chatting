# ⌨ Spring-SSE-Chatting ⌨  
Spring과 SSE 프로토콜을 활용한 익명 채팅 웹사이트입니다  
닉네임과 원하는 채팅방 번호를 입력하여 로그인하며 해당 번호의 채팅방에서 익명의 다수와 대화를 할 수 있습니다

## 💻 Introduce Program 💻

### ✔ Preview

> #### ◼ 입장하기
>> 채팅방에 입장하기 위해서는 닉네임과 입장할 채팅방 번호가 필요합니다    

> #### ◼ 채팅하기
>> 해당 채팅방 번호 내 익명의 다수와 채팅을 할 수 있습니다
